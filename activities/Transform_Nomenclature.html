Task 2 – Transform Nomenclature

This task consists of three exercises focused on understanding transformation nomenclature and frame relationships using MATLAB.

1. A vector AP is rotated about the YA axis by an angle of 45 degrees, and then rotated about the XA axis by an angle of 60 degrees. Give the rotation matrix that performs these rotations in the indicated order.

Instructions

  • Rotated the vector about the YA axis by 45 degrees.
  • Rotated the vector about the XA axis by 60 degrees.
  • Define the transformation matrix.
                            
            clc;
            clear;

            thetaY = (45*pi/180);
            thetaX = (60*pi/180);

            Ry = [ cos(thetaY) 0 sin(thetaY);
            0 1 0;
            -sin(thetaY) 0 cos(thetaY) ];

            Rx = [ 1 0 0;
            0 cos(thetaX) -sin(thetaX);
            0 sin(thetaX) cos(thetaX) ];

            R = Rx * Ry
                            
                    
    Result 
                
        0.7071         0    0.7071
        0.6124    0.5000   -0.6124
        -0.3536    0.8660    0.3536

2. The reference frame B is rotated with respect to A about the XA axis by 30 degrees. The translation of B from A is given by the vector: [5 10 0]^T Formulate the homogeneous transformation matrix.

Instructions

  • B rotation with respect to A
  • Formulate the homogeneous transformation matrix.
                            
            clc;
            clear;
            
            thetaX = (30*pi/180);
            
            Rx = [ 1 0 0;
            0 cos(thetaX) -sin(thetaX);
            0 sin(thetaX) cos(thetaX) ];
            
            d = [5;
            10;
            0];
            
            T = [ Rx d;
            0 0 0 1 ];
            
            T
                            
                    
        Result 
                    
            1         0        0       0
            0      0.8660      0      10 
            0      0.5000   0.8660     0
            0         0     0.3536     1

  • The value of ATC
